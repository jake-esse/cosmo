## Common Typography Patterns

### Large Headings (60px)
```css
font-family: Crimson Text;
font-size: 60px;
line-height: 0.9;
letter-spacing: -1.8px;
color: black;
```

### Body Text (15px)
```css
font-family: DM Sans;
font-size: 15px;
line-height: 1.4;
letter-spacing: -0.075px;
color: #6F6F6F;
```

### Buttons (14px)
```css
font-family: DM Sans;
font-weight: bold;
font-size: 14px;
letter-spacing: -0.35px;
```
